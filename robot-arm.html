Projects >> FTobler Robotic Arm

Author: Nemi Rai

Date: January 2020

Summary

This robotic arm is a community open source project based off an original design by Florin Tobler. The arm is fully 3D-Printed, with the exception of the electronic parts. A RAMPS 1.4 board controls the arm by driving 3 Nema 17 stepper motors for each axis of the arm; the board also drives a BYJ48 stepper motor for the arm's gripper.

The robotic arm's firmware, as well as other files, can be found on the GitHub. G-code is used to program the arm's movements in either relative or absolute coordinates. In the early stages of the project the interface was very limited. I wrote a Python program to read g-code from a file and deliver it over serial to the Arduino Mega 2560 microcontroller to control the arm. The software for controlling the arm has since been updated as seen on the project website.

Parts list
  • 3x Nema 17 stepper motor
  • Arduino Mega 2560 microcontroller
  • RAMPS 1.4 Board
  • 12V 4A power supply

Future additions.

The community has added many features to this arm since the start of the project. Version 1.0 of the arm uses 3D-printed gears that are in direct contact with each other for each of the axis. Because of the limitations of the resolution with FDM 3D-printing, having the gears in direct contact with each other was not ideal because the arm would sometimes seize or not move smoothly.

Some additional features that the community has produced include:

  • A belt-driven track based off 4040 V-extrusions and an additional Nema 17 stepper motor.
  • Fully converting the arm to a fully belt-driven model.
