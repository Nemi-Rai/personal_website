Projects >> 3-Axis Robotic Arm

Author: Nemi Rai

Date: January 2020

Summary

This robotic arm is a community open source project based off an original design by Florin Tobler. The arm is fully 3D-Printed, with the exception of the electronic parts. The arm is controlled by a RAMPS 1.4 board which drives 3 Nema 17 stepper motors for each axis of the arm; the board also drives a BYJ48 stepper motor for the arm's gripper.

Parts list
  • 3x Nema 17 stepper motor
  • Arduino Mega 2560 microcontroller
  • RAMPS 1.4 Board
  • 12V 4A power supply
