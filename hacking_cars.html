Hacking cars

Author: Nemi Rai

Date: June 2022

Summary

Vehicles are becoming increasingly complicated over time with more components communicating with each other through electrical means rather than purely mechanical. Vehicles benefit from security through obscurity because they are very complicated systems. However, like all systems they are not 100% secure and can be hacked.

Attack Surface and Initialization Vectors

Whenever a vehicle has a feature where two or more components react to each other, such as adaptive headlights, there must be an element facilitating the communication between the components. Adaptive headlights adjust with the steering wheel, so there must be some element that facilitates the communication between the steering wheel and headlight assembly. The ECU is usually responsible for most communications, although vehicles also do use shared and dedicated busses for inter-vehicle communications.

Accessing a vehicle's ECU is near-impossible, however modern cars usually have an in-car-entertainment (ICE) or infotainment system which is a good starting point for a researcher to look for vulnerabilities. The ICE system can use wireless protocols such as bluetooth and others, which makes it possible for an intruder to gain access remotely and possibly control ICE connected systems.
