Projects >>> Pi-Hole

Author: Nemi Rai

Date: August 2021

Summary

Pi-Hole is a DNS sinkhole that blocks advertisements network-wide. The software is run on Raspberry Pi microcomputers and works by not forwarding any web requests to known advertisement service domains. Pi-Hole also has standard DNS services such as caching using dnsmasq and DNS forwarding.
