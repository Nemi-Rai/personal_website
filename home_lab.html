Home Lab

Author: Nemi Rai

Date: June 2022

Summary

A Cybersecurity home lab is an isolated computer environment where a researcher can safely run tests on computer systems and networks. There are many reasons why a researcher would want to run tests in a home lab. When setting up a remote server for the first time, a home lab clone provides a development environment to test out new services that an administrator may want to install on the live server.

The main reason for a home lab is to safely test systems for vulnerabilities in an isolated environment with no legal risk. Performing the same actions on a system that does not belong to you without permission has serious legal consequences.

Components of a Home Lab

A home lab consists of an attacker machine and other end systems connected by switches and routers. The attacker machine is usually linux-based and Kali is the distribution of choice as it comes with many penetrations testing tools preinstalled. Any linux distribution can be used for the attacker machine because these tools can be installed on any linux distribution with their package manager.
