Projects >> MacEwan University Pi Robot "MUPiBot"

Author: Nemi Rai

Date: July 2021

Summary

MUPiBot is a multi-sensor track based robot that uses a Raspberry Pi 3B+ micro comupter and an Arduino Nano micro controller to sense and navigate it's environment.
